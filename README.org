#+title: PureScript Free Turtle interpreter

A demonstration of a Turtle interpreter based on the Free monad, with HTML Canvas graphics support.

[[file:img/two-stars.png]]

#+BEGIN_SRC purescript
main :: Context2DEff
main = renderTurtleProgOnCanvas "turtleCanvas" $ do
  star
  penUp
  forward 40.0
  left 100.0
  penDown
  color Red
  star

star = do
  right 144.0
  forward 100.0
  right 144.0
  forward 100.0
  right 144.0
  forward 100.0
  right 144.0
  forward 100.0
  right 144.0
  forward 100.0
#+END_SRC

*  Usage

In your shell, type:

: npm install
: npm run build
: npm run bundle

Then point your browser to [[./html/index.html]].
